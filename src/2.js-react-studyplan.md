- 22~23: 문자열 조작

**🔍 React 보완**

- State, Props에서 JS 변수 개념 그대로 사용

**📌 학습 목표**

- 변수와 데이터 타입 구분, 적절한 자료형 선택
- 문자열/숫자 연산 자유롭게 활용

**💡 SaaS 적용 사례**

- 창업자 희망 보증금(숫자)과 매물 보증금 비교
- 상태 값(진행중/계약완료) 필터링 시 논리 연산자 활용

## 2. 배열 & 객체

**🎯 JS 강의 추천 (섹션 3)**

- 25~33: 배열 생성, 접근, 수정, 메서드(push, filter, map, find)
- 객체 생성, 프로퍼티 접근/수정
- 중첩 객체 & 배열

**🔍 React 보완**

- State로 배열/객체 다루기
- JSX에서 map()으로 UI 렌더링

**📌 학습 목표**

- 배열과 객체를 자유롭게 조작
- map, filter, reduce 패턴 익히기

**💡 SaaS 적용 사례**

- 매물 목록에서 조건에 맞는 매물만 filter
- map()으로 매칭 결과 리스트 출력

## 3. 함수 & 스코프

**🎯 JS 강의 추천 (섹션 4)**

- 35~48: 함수 선언, 매개변수, 반환값
- 스코프(전역, 지역), 클로저

**🔍 React 보완**

- onClick, onChange 등 이벤트 핸들러 작성
- 커스텀 훅 작성 시 함수 재사용 구조

**📌 학습 목표**

- 재사용 가능한 함수 작성
- 스코프 이해로 변수 충돌 방지

**💡 SaaS 적용 사례**

- getFilteredProperties()로 조건 필터링 로직 재사용

## 4. 조건문 & 반복문

- 49~55: if, else, switch

**🔍 React 보완**

**📌 학습 목표**

- 조건에 따라 코드 흐름 제어

**💡 SaaS 적용 사례**

- 상태값에 따라 버튼 숨김/표시

**💡 SaaS 적용 사례**
